Perpetual vs Periodic Inventory Systems

Understanding the Two Methods of Tracking Inventory

Perpetual Inventory System

Definition: Continuously updates inventory records after each purchase and sale

  • Real-time inventory tracking
  • Uses "Inventory" account for all transactions
  • COGS calculated at each sale
  • Common in modern retail (barcode scanners)
  • More expensive but more accurate
  • Requires sophisticated software
Key Journal Entries:
Purchase: Debit Inventory
Sale: Debit COGS, Credit Inventory
Periodic Inventory System

Definition: Updates inventory only at end of accounting period through physical count

  • Inventory tracked periodically (monthly, yearly)
  • Uses "Purchases" account during period
  • COGS calculated at period end
  • Common in small businesses
  • Less expensive, simpler system
  • Requires physical counts
Key Journal Entries:
Purchase: Debit Purchases
Sale: No inventory entry until period end
